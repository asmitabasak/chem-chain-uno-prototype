## ðŸ§  Core Gameplay Mechanics
1. **The Reaction Link (Power Move):** Play $Na$ on $Cl$ to form a bond. This triggers a visual animation and forces opponents to draw cards.
2. **The Power Clash:** When two players try to react, the **Reactivity Series** decides the winner (e.g., $F$ beats $Cu$).
3. **The Noble Gas Shield:** Play $Ar$ or $Ne$ to block all incoming reactions and reset the turn.
4. **Special Cards:** Includes Catalysts (Wilds), Inhibitors (Skip), and Equilibrium (Reverse).
